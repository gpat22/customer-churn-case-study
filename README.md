# customer-churn-case-study
Customer Churn Analysis Case Study

Business Problem
A telecommunications company is experiencing customer loss (churn), leading to reduced revenue and increased acquisition costs. The objective of this project is to identify the factors contributing to customer churn and recommend strategies to improve customer retention.

Objectives
Analyze customer churn behavior.
Identify key drivers of churn.
Create an interactive dashboard.
Provide actionable recommendations.

Dataset
Source: IBM Telco Customer Churn Dataset (Kaggle)
Records: 7,043
Features: 21

Tools Used
Python
Pandas
NumPy
Matplotlib
Tableau
Google Colab Notebook

Key Findings
Month-to-month contract customers have the highest churn rate (42.7%).
Customers using Fiber Optic internet show higher churn (41.9%).
Customers without Tech Support are more likely to churn (41.7%).
Electronic Check users have the highest churn rate (45.3%).

Recommendations
Promote long-term contracts through incentives.
Bundle Tech Support with internet plans.
Improve customer experience for Fiber Optic users.
Encourage automatic payment methods.

Future Improvements
Build a churn prediction model.
Perform customer segmentation.
Deploy an interactive dashboard online.
